# Список задач (Project Tasks)

## Актуальные задачи:
Необходимо сделать так, чтобы приложение смогло запускаться на других компьютерах. На друих компьютерах может не быть моей БД programms. Необходимо будет её создать.

## Важное уточнение по реализации:
Делать максимально просто! Весь SQL-код пишем прямо внутри функций кнопок (в методах класса окна), чтобы студентам было видно: "нажал кнопку — выполнился этот SQL-запрос".
